# 数据处理复现资料

[中文](README.md) | [English](README.en.md)

本目录保存 DeepSDF 数据预处理相关的历史资料，不是主复现入口。基础飞机实验请
先阅读仓库根目录 `README.md`；TTT/LoRA 扩展实验请阅读
`ttt_deepsdf/README.zh-CN.md`。

## 适用场景

需要以下信息时再阅读本目录：

- 了解如何把 ShapeNet 原始 mesh 转换成 DeepSDF 的 SDF 采样数据。
- 在 CPU-only 或无显示器 Linux 服务器上复现数据预处理流程。
- 追溯项目使用过的外部文章、数据集说明和离线网页备份。
- 原始网页不可访问时，查看已归档的本地备份。

如果只做推理、可视化或使用已准备好的训练数据，通常不需要先阅读本目录。

## 推荐阅读顺序

1. 需要复现数据预处理时，先读 `DeepSDF_CPU_服务器复现全流程.md`。
2. 需要核对外部资料时，打开 `weixinhum-149218796-link.txt`。
3. 只有在需要离线查看旧网页时，再打开 HTML 备份。
4. 如果 HTML 备份出现乱码，以原始 URL 为可靠来源。

## 隐私说明

复现手册不发布真实 IP、私钥路径、用户名或机器绝对路径。实际运行时，请用自己
环境中的地址、账号和目录替换文档中的占位符。
