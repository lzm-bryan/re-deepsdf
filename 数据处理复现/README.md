# 数据处理复现目录使用说明

本目录是**历史数据处理资料和参考链接归档**，不是主复现入口。

如果你是第一次阅读本仓库，建议先看根目录 `README.md` 跑通 DeepSDF 飞机复现；如果关注 TTT/LoRA 实验，直接看 `ttt_deepsdf/README.md`。本目录适合在需要追溯数据来源、查看 CPU/服务器处理记录时再阅读。

## 什么时候需要看这里

- 想了解早期数据处理和服务器复现流程。
- 想找当时参考过的数据集说明或网页链接。
- 想检查离线网页备份是否还可用。

一般情况下，跑模型和看结果不需要先阅读本目录。

## 推荐使用顺序

1. 需要流程说明时，先看 `DeepSDF_CPU_服务器复现全流程.md`。
2. 需要追溯外部资料时，看 `weixinhum-149218796-link.txt`。
3. 需要离线查看旧网页时，再尝试打开 `weixinhum-149218796-mirror.html` 或 HTML 备份。
4. 如果 HTML 出现乱码，以 txt 中的原始链接为准。
